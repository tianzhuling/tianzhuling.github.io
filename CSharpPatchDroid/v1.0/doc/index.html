作品简介

一句话概括

CSharpPatchDroid 是一个运行在安卓系统上的.NET逆向工具

详情

这是一款免费且集成反编译，回编译等功能，为在安卓上逆向.NET工程而量身打造的安卓软件CSharpPatchDroid。 内部采用命令行模式，较为直观。支持UnityC#代码逆向与修改，简单.NET程序反编译。 自动补全依赖，自动匹配C#与.NET版本等功能

使用方法

整体用法

本软件用非传统命令行的方式与用户交互，初始化好后，提示用户输入命令，输入完命令后，提示填写参数，部分参数可留空，填写完参数后便会运行

参数命令

参数命令名 功能 版本支持
/over 立即结束当前的参数填充，将剩下参数都设置为空 v1.0引入
/skip 跳过当前参数填充，到下一个参数填充，相当于直接回车 v1.0引入
/quit 用于在参数填充时，临时撤销该命令 v1.0引入

命令

命令 功能 参数 版本
create 用于创建一个反编译项目 必填"dll路径", "C#版本", "框架(.NETCOREAPP,.NETSTANDARD,.NETFRAMEWORK,SILVERLIGHT)默认.NETFRAMEWORK", "框架版本", "项目名称", "输出根路径(默认在/storage/emulated/0/CSharpPatchDroidOutput/projects)" v1.0
sethome 用于设置HOME路径(已弃用) 必填"新HOME路径" 测试版本引入，v1.0弃用
homepath 用于获取HOME路径 无参数 v1.0
updateproject 用于回编译项目而不指定.csproj位置，现已停止维护该命令，取而代之的是update 必填"项目名称", "根目录路径(HOME目录，默认/storage/emulated/0/CSharpPatchDroidOutput/projects)" v1.0引入，v1.0版本不再维护
update 用于回编译项目 必填"项目名称", "根目录路径(HOME目录，默认/storage/emulated/0/CSharpPatchDroidOutput/projects)", ".csproj文件路径(默认自动查找)" v1.0
clear 清除/storage/emulated/0/CSharpPatchDroidOutPut的所有内容 三次确认Y/n v1.0
在本表格参数前有"必填"的为必填参数，其余皆为选填参数

概念

  1. HOME路径，默认/storage/emulated/0/CSharpPatchDroidOutPut/projects，反编译项目的通用根目录，可通过修改/storage/emulated/0/CSharpPatchDroidOutPut/settings.json来修改
  2. 日志相关：
    1. 普通日志输出，除编译阶段的输出，路径在/storage/emulated/0/CSharpPatchDroidOutPut/log.txt
    2. 构造日志输出，路径在编译阶段的日志输出，路径在/storage/emulated/0/CSharpPatchDroidOutPut/buildLog.txt
